## Bonus Screens (additional UX notes)

### Tutorial / Learnability
![Tutorial overlay](assets/04-tutorial-overlay.png)

**Quick note:** The overlay is informative but can cover key UI elements and interrupt task completion.  
**Suggestion:** Add “Don’t show again”, break content into shorter steps, and use contextual tooltips.

### VIP / Store Clarity
![VIP perks](assets/05-vip-perks.png)

**Quick note:** Value proposition can be clearer at a glance.  
**Suggestion:** Summarize benefits upfront, reduce visual clutter, and provide a simple comparison view.
